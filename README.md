# 🔐 EncMaster: Military-Grade AES-256 File & Folder Encryption

EncMaster is a **free, open-source privacy tool** designed to secure your sensitive data offline. Whether you are on Linux, Windows, or Mobile, EncMaster ensures your files stay private with industry-standard encryption.

## 🚀 Key Features
*   **True AES-256 Security:** Your files are locked with the same math used by governments.
*   **Secure Shredder:** Permanently deletes original files so they can't be recovered.
*   **Offline First:** No data ever leaves your device. No cloud, no risk.
*   **Multi-Platform:** Use the Desktop app on Linux or our [Web Hub](https://github.io) for Mobile & Windows.

## 🛠️ Installation (Linux)
1. Download the `EncMaster_Linux` binary from this repository.
2. Open terminal and run: `chmod +x EncMaster_Linux`
3. Launch with: `./EncMaster_Linux`
